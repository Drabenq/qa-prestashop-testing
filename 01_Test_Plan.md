# TEST PLAN
Proyecto: Testing página Ofertas PrestaShop
QA: Hernán Machado
Fecha: 22/02/2026

---

## 1. Objetivo

Validar el correcto funcionamiento funcional y no funcional de la página de ofertas desarrollada en PrestaShop, asegurando que los usuarios puedan visualizar productos, navegar correctamente y realizar acciones sin errores.

---

## 2. Alcance

Se testearan:

- Visualización de productos
- Navegación entre enlaces
- Botones de acción
- Formularios
- Validaciones
- Comportamiento responsive
- Rendimiento básico

No se testeará:

- Integración con pasarelas de pago reales
- Backend administrativo
- Base de datos interna

---

## 3. Tipos de prueba

- Pruebas funcionales
- Pruebas de regresión
- Pruebas de usabilidad
- Pruebas responsive
- Pruebas exploratorias
- Pruebas básicas de rendimiento

---

## 4. Entorno de prueba

Sistema Operativo: Windows 10
Navegador principal: Google Chrome
Resolución: 1920x1080
Testing Mobile: Chrome DevTools (iPhone 14 simulation)

---

## 5. Criterios de entrada

- Página accesible públicamente
- Sin mantenimiento activo

---

## 6. Criterios de salida

- Todos los test cases ejecutados
- Bugs documentados en Jira
- Evidencias adjuntas
